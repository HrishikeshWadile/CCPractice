1. Open Azure Cloud Shell

Go to:
👉 https://portal.azure.com

Click:

Cloud Shell → Select Bash

2. Set variables (optional but clean practice)
storageName=mystorageacct$RANDOM
resourceGroup=MyRG
location=centralindia
3. Create Resource Group
az group create \
  --name $resourceGroup \
  --location $location
4. Create Storage Account
az storage account create \
  --name $storageName \
  --resource-group $resourceGroup \
  --location $location \
  --sku Standard_LRS
5. Enable Static Website Hosting
az storage blob service-properties update \
  --account-name $storageName \
  --static-website \
  --index-document index.html \
  --404-document error.html
6. Create simple web files
index.html
echo "<h1>Hello from Azure Static Website</h1>" > index.html
error.html
echo "<h1>Page not found</h1>" > error.html
7. Upload files to Azure Storage
az storage blob upload \
  --account-name $storageName \
  --container-name '$web' \
  --file index.html \
  --name index.html \
  --auth-mode login
az storage blob upload \
  --account-name $storageName \
  --container-name '$web' \
  --file error.html \
  --name error.html \
  --auth-mode login
8. Get Website URL
az storage account show \
  --name $storageName \
  --resource-group $resourceGroup \
  --query "primaryEndpoints.web" \
  --output tsv
🌐 Output

Open the URL shown in terminal:

https://<storage-name>.z13.web.core.windows.net/

You will see:

Hello from Azure Static Website
🧾 Result

Static web application successfully deployed using Azure Storage static website hosting.

📌 Conclusion

Azure Blob Storage static website feature provides S3-like hosting for static web applications without needing servers.
